# API Reader Skill

Reads any REST API given a base URL and optional API key or Bearer token. Fetches JSON from paths or discovers endpoints.

## Quick start

```bash
.venv/bin/python .cursor/skills/api-reader/scripts/api_reader.py <BASE_URL> --bearer TOKEN --path /api/tables --output out.json
```

See [SKILL.md](SKILL.md) for full usage and arguments.
